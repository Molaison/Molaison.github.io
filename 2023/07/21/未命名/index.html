在Git中，git log命令用于查看提交历史。通过指定<commit_range>参数，您可以限定git log命令的输出范围，只显示符合条件的提交。

<commit_range>参数可以采用以下不同的格式：

  1. 单个提交（Commit）: 输入提交的完整哈希值（SHA-1）或其缩略形式，将仅显示该提交的信息。

  2. 分支或标签名: 您可以输入分支名称或标签名称来查看该分支或标签的提交历史。

  3. 提交范围: 使用双点 .. 指定提交范围，从一个提交到另一个提交。

    这将显示从<commit_start>（不包括）到<commit_end>（包括）之间的提交。

  4. 向后指定提交数目: 使用 -<number> 指定要显示最近多少个提交。

    例如，git log -5 将显示最近的5个提交。

  5. 时间范围: 使用 --since 和/或 --until 参数来指定时间范围内的提交。

    其中<date>是日期时间的格式，例如："2023-01-01"。

  6. 作者过滤: 使用 --author 参数按作者筛选提交。
